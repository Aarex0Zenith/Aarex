// roadmap

THE JOURNEY

How to become an ethical hacker from absolute zero — the honest roadmap, no shortcuts, no bullshit.

STEP 01
LINUX FUNDAMENTALS

Start with Linux. Everything in cybersecurity runs on it. Learn file systems, permissions, bash scripting, process management, and networking commands. Kali Linux is your playground.

  • Install Kali/Ubuntu in VirtualBox
  • Master: ls, grep, find, chmod, netstat, nmap
  • Write 10 bash scripts
STEP 02
NETWORKING DEEP DIVE

Without networking knowledge, you're blind. TCP/IP, OSI model, DNS, HTTP/HTTPS, ARP, routing — these are the bones of every hack.

  • CompTIA Network+ as reference
  • Practice with Wireshark & Packet Tracer
  • Understand how websites actually work
STEP 03
PROGRAMMING

Python is your weapon. Start simple — automate tasks, write port scanners, build tools. Then touch JavaScript to understand web exploitation.

  • Python: requests, socket, subprocess, scapy
  • Build 5 security tools from scratch
  • Read others' exploit code on GitHub
STEP 04
WEB SECURITY

OWASP Top 10 is your bible. Learn each vulnerability category: SQLi, XSS, IDOR, SSRF, CSRF, file upload bypasses. Practice on DVWA and PortSwigger Labs.

  • Set up Burp Suite Pro workflow
  • Complete all PortSwigger Web Security Academy labs
  • Read every HackerOne disclosed report
STEP 05
CTF PRACTICE

CTFs (Capture The Flag) are where theory becomes skill. Solve challenges in web, crypto, forensics, pwn, reverse engineering. Lose often. Learn every time.

  • TryHackMe — 100 rooms minimum
  • HackTheBox — 20+ machines
  • PicoCTF, CTFtime.org weekly
STEP 06
BUG BOUNTY

Turn your skills into real impact and money. Start with private programs, read scope carefully, report responsibly. One good P1 changes your whole resume.

  • HackerOne, Bugcrowd, Intigriti accounts
  • Focus on one target for 30 days
  • Document everything, write clear reports
STEP 07
ADVANCED SECURITY RESEARCH

This is the frontier. CVE research, malware analysis, red teaming, cloud security, binary exploitation. Build tools, publish writeups, create your reputation.

  • Certifications: OSCP, CEH, eJPT
  • Build a public GitHub portfolio
  • Write security blog posts & research papers
